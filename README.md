# KaziConnect

A multi-channel freelance platform designed for the African gig economy that connects freelancers and clients through AI-powered job matching, a web-based productivity dashboard, and USSD/SMS access.

## Features
- **AI Job Matching**: Automatically computes match scores based on required freelance skills.
- **USSD/SMS access**: Supports basic feature phones via Africa's Talking API.
- **Rich Dashboard**: Modern web interface using Vite + React.

## Setup

```bash
# Start Backend
cd backend
npm install
node index.js

# Start Frontend
cd frontend
npm install
npm run dev
```
